# StickerVault

An anime sticker e-commerce platform for discovering, collecting, and trading vibrant anime stickers.

## Tech Stack

- **Frontend**: Next.js 16 + React 19 + TailwindCSS 4
- **UI Components**: Radix UI + shadcn/ui
- **Animations**: Framer Motion
- **State Management**: TanStack React Query
- **Forms**: React Hook Form + Yup
- **Internationalization**: i18next
- **Monorepo Tooling**: Nx

## Getting Started

### Prerequisites

- Node.js (see `.nvmrc` for recommended version)
- pnpm

### Install

```bash
pnpm install
```

### Run Development Server

```bash
pnpm dev
```

### Build for Production

```bash
pnpm build
```

### Lint

```bash
pnpm lint
```

## Features

- Anime sticker themed UI with vibrant pink/magenta palette
- Hero section showcasing the StickerVault brand
- Internationalization support (i18next)
- Responsive layout with dark mode support
- Component library built on Radix UI primitives
